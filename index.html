01 · Quem sou

Sobre mim

Sou desenvolvedor web com foco em criar experiências digitais que unem design, performance e usabilidade. Minha jornada começou com a curiosidade de entender como os sites funcionam por trás das telas, e hoje me dedico a transformar ideias em código funcional e bonito.

Tenho experiência com HTML5, CSS3, JavaScript, Bootstrap, Responsividade e estou sempre explorando novas tecnologias para aprimorar minhas habilidades. Acredito que a tecnologia deve ser acessível e que cada projeto é uma oportunidade de aprender algo novo.

Atualmente foco em front-end e na construção de interfaces que proporcionam experiências incríveis para os usuários.

04 · Versionamento

Git & GitHub

Controle de versão é essencial no desenvolvimento moderno. Aqui estão minhas práticas e aprendizados.

Comandos essenciais

  • $ git init
  • $ git add .
  • $ git commit -m "mensagem"
  • $ git push origin main
  • $ git pull
  • $ git branch & git merge

Workflow com GitHub

Utilizo o GitHub para hospedar meus repositórios, colaborar em projetos open-source e manter um histórico claro das alterações. Meus principais repositórios estão organizados com README, .gitignore e branches bem definidas.

Práticas: commits semânticos, pull requests e revisão de código.
